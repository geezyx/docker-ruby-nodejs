Docker-Ruby
=======================

Small bootstrap container for a ruby app.  Provide the environment variables below and off you go!

Environment variables
---------------------

* `GIT_URL`
  - Git URL for cloning
* `PROJECT_PATH`
  - Project path root (e.g. `/var/myapp`)
* `START_CMD`
  - Command to initialize service (e.g. `ruby myscript.rb`)
